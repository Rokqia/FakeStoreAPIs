# Fake Store APIs Testing using Postman

This project focuses on testing Fake Store APIs to ensure reliability and effectiveness using Postman.

## Setup

1. Clone this repository to your local machine.
2. Import the provided Postman collection (`Fake_Store_APIs.postman_collection.json`) into Postman.
3. Set up the environment variables if necessary, such as `BaseURL`, `LimitValue`, etc.

## Testing

- Implemented rigorous testing for each endpoint to ensure reliability and functionality.
- Utilized Postman's testing capabilities to validate response data, status codes, and response times.

## Environment Variables

Ensure the following environment variables are configured in Postman:

- `BaseURL`: Base URL of the Fake Store API.
- `LimitValue`: Limit value used in testing endpoint limits.
